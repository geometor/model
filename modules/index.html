geometor.model¶

The geometor.model module provides the core data structures and logic for constructing geometric models in 2D space. It serves as the foundation for the Geometor project, enabling the creation, manipulation, and analysis of geometric constructions.

Key Components:¶

  • Model: The central class Model representing a collection of geometric elements.

  • Elements: Wrappers around SymPy geometry objects (Point, Line, Circle, Polygon).

  • Mixins: Modular functionality for the Model class.

  • Analysis: Tools for analyzing geometric relationships (Section, Chain, Wedge).

Usage:¶

Initialize a Model and use its methods to add points, construct lines and circles, and perform geometric operations.

class geometor.model.Chain(sections: list[geometor.model.sections.Section])¶

A class representing a chain of connected golden sections, facilitating the extraction of segments, points, and lengths, as well as analyzing the flow and symmetry within the chain.

Each chain’s flow is characterized by the comparative lengths of consecutive segments, represented symbolically to understand the progression and transitions in segment lengths. Furthermore, this module empowers users to explore symmetry lines within chains, unveiling a subtle, profound aspect of geometric harmony.

Retrieves the IDs of the ancestors for the given element.
