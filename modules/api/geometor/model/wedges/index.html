geometor.model.wedges¶

The geometor.model.wedges module provides wedge construction and manipulation for the Model class.

Classes¶

Mixin for the Model class containing wedge construction operations.

Mixin for the Model class containing wedge construction operations.

set_wedge(pt_center: sympy.geometry.Point, pt_radius: sympy.geometry.Point, pt_sweep_start: sympy.geometry.Point, pt_sweep_end: sympy.geometry.Point, direction='clockwise', classes: list = None, ID: str = '') → Wedge[source]¶

sets a Wedge from 3 points and adds it to the model.

operations¶

  • create an instance of geometor.model.Wedge

  • create a details object from Element

  • add parents to details

    initial parents are the two starting points

  • check for duplicates in in the model

  • find intersection points for new element with all precedng elements

  • Add circle to the model.

type - pt_radius:

sympy.geometry.point.Point : point to mark radius

param - pt_end:

type - pt_end:

sympy.geometry.point.Point : A SymPy Point marking the sweep of the wedge

type - classes:

list optional : A list of string names for classes defining a set of styles. Defaults to None.

param - ID:

type - ID:

str optional : A text ID for use in plotting and reporting. Defaults to an empty string.

returns:

The portion of a circle

rtype:

  • Wedge

Example

>>> from geometor.elements import *
>>> model = Model("demo")
>>> A = model.set_point(0, 0, classes=["given"], ID="A")
>>> B = model.set_point(1, 0, classes=["given"], ID="B")
>>> model.construct_circle(A, B)
>>> model.construct_circle(B, A)
>>> model._set_wedge_by_IDs('A', 'B', 'C')
<Wedge object ...>

Notes

SymPy defines a circle as a center point and a radius length, so the radius length is calculated for the spg.Circle.
