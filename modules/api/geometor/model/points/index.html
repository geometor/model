set_point(x_val: sympy.Expr, y_val: sympy.Expr, parents: list = None, classes: list = None, ID: str = '', guide: bool = False) → sympy.geometry.Point¶

Adds a point to the model, finds duplicates, cleans values, and sets parents and classes.

Parameters:

  • x_val (-)

  • y_val (-)

  • parents (-) – Defaults to None.

  • list (- classes) – a set of styles. Defaults to None.

  • optional (A text ID for use in plotting and) – a set of styles. Defaults to None.

  • str (- ID) – reporting. Defaults to an empty string.

  • optional – reporting. Defaults to an empty string.

Returns:

- :class:`sympy.geometry.point.Point`

Return type:

The set point.

Example

>>> from geometor.model import *
>>> model = Model("demo")
>>> model.set_point(0, 0, classes=["given"])
<spg.Point object ...>

Notes

The function simplifies the x and y values before adding, and it updates the attributes if the point is already in the model.
